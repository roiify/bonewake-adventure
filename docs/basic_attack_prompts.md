# Basic-Attack Animation Prompts

Ready-to-paste PixelLab `action_description` strings for the basic-attack animation of every hero and enemy. Use with `animate_character(..., animation_name="attack", frame_count=8, directions=["east"])`.

Style rules: focus on the motion only (no environment, no flavor text). Each prompt is one clean action that resolves in 8 frames. Hero attacks emphasize their signature weapon/style. Enemy attacks emphasize how they actually reach the player — bite, claw, swing, lunge.
